# IntelliSearch 更新日志 - 2025-03-10

## 核心更新

### 1. 错误处理系统标准化 ✨

**新增模块：**

- `core/error_center.py` - 统一错误中心处理器
- `core/exceptions.py` - 标准化异常类和错误码注册系统

**主要特性：**

- **自定义异常标准化**：所有异常继承自 `IntelliSearchError`，统一接口
- **错误码去重验证**：通过 `register()` 函数注册，防止重复（如 `MCP_CONNECTION_ERROR`）
- **错误码注册机制**：自定义，自动验证唯一性
- **统一UI展示**：`UIEngine.display_exception()` 集成错误中心，自动发送UI提示

**使用示例：**

```python
from core.error_center import error_center
from core.exceptions import IntelliSearchError

# 自动处理和显示
try:
    risky_operation()
except IntelliSearchError as e:
    decision = error_center.handle(e)
    error_center.display_error(e)
```

### 2. UI系统组件化重构 🎨

**核心改进：**

- **组件化架构**：`core/UI/components/` 目录下独立组件（Input, Select, Confirm, Display等）
- **唯一Live单例**：`live.py` 统一管理 Live 实例，避免多个 Live 冲突
- **交互组件**：支持 TUI 交互模式（选择、确认、输入）
- **Rich渲染**：所有组件支持 Rich 格式化和样式

**组件列表：**

- `InputControl` - 输入组件（支持验证、密码、长度限制）
- `SelectControl` - 选择组件（单选/多选）
- `ConfirmControl` - 确认组件（Yes/No选择）
- `DisplayControl` - 显示组件（支持多种消息样式）
- `WelcomeUI` / `ToolCallUI` - 专用UI组件
  **注意：**内部组件更加推荐直接使用rich，不必引入UI系统，UI系统的作用主要是给mcp_server提供UI自由度的

---

### 3. 清除Hack实现 🧹

**文件操作模块重构：**

- **移除 security 依赖**：`mcp_server/operate_file/*` 移除所有权限验证代码
- **直接使用 Path**：改用 `pathlib.Path.resolve()` 处理路径
- **统一异常处理**：所有异常返回错误字符串，不再返回 `None`

**清除的Hack代码：**

- ❌ 散落在各处的 `"access denied"` 字符串判断
- ❌ `validate_path()` 权限验证
- ❌ `SecurityError` 异常捕获和 `pass`
- ❌ 权限配置文件依赖

**影响文件：**

- `mcp_server/operate_file/list_ops.py`
- `mcp_server/operate_file/read_ops.py`
- `mcp_server/operate_file/write_ops.py`
- `mcp_server/operate_file/manage_ops.py`

## 迁移指南

### 错误处理

```python
# 旧方式
try:
    operation()
except Exception as e:
    print(f"Error: {e}")

# 新方式
from core.error_center import error_center
from core.exceptions import IntelliSearchError

try:
    operation()
except IntelliSearchError as e:
    error_center.handle(e)
```

### UI交互

```python
from core.UI.ui_engine import UIEngine

# 输入
name = UIEngine.prompt_input(message="请输入姓名", min_length=2)

# 选择
color = UIEngine.prompt_select(message="选择颜色", options=["红", "蓝", "绿"])

# 确认
confirmed = UIEngine.prompt_confirm(message="确定删除？")

# 显示
UIEngine.display("操作成功", style="success")
```
