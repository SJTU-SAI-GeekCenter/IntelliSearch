# 微信公众号文章处理

在接下来的任务中，你需要处理如下的微信公众号文章，你的具体要求如下：

## 获取核心内容

1. 将HTML转换为**人类可读的纯文本形式**，删除所有HTML标签，**不要带任何的 HTML 和 Markdown 标签**！
2. 文本清洗：
   - 删除所有转义字符（如\n、\t等）
   - 删除无意义字符和乱码
   - 删除"文字："、"排版："、"摄影："及其后面的内容
   - 删除"长按关注公众号"、"关注我们"等推广内容
   - 删除文章结尾的广告和推广信息
3. 文本优化：
   - 添加合适的标点符号
   - 修复缺失的标点
   - 将英文引号转换为中文引号（如'转换为「）
4. 生成优化后文章摘要

注意！不要随意删除文章中有意义的信息！

## 输出格式

你需要严格遵循如下的输出格式：

<content>
处理好的文本
</content>

<summary>
对应的摘要
</summary>
